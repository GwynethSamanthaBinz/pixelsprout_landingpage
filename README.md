Eine moderne, responsive Landingpage für **PixelSprout** – deine Pixel-Art Goal-Management-App!

## 📋 Features

✨ **Playful & Colorful Design** - Farbenfrohe Aesthetik mit Pixel-Art-Vibes
📱 **Vollständig Responsiv** - Funktioniert auf allen Geräten
🎮 **Interaktive Elemente** - Smooth Scrolling und Hover-Animationen
🚀 **Optimiert** - Schnelle Ladezeiten, moderne CSS
⚙️ **Einfach zu Customizen** - Klare Struktur, kommentierter Code

1. **Datei öffnen**: `index.html` im Browser öffnen

3. **Deployen**: Auf einen Web-Server hochladen (z.B. Netlify, Vercel, GitHub Pages)

## 🎯 Customization-Tipps

### Farben ändern
In `css/styles.css` die CSS-Variablen anpassen:
```css
:root {
    --primary-color: #FF6B9D;      /* Ändere die Farbe hier */
    --secondary-color: #C44569;
    /* ... weitere Farben ... */
}
```

### Text/Inhalte ändern
In `index.html` die entsprechenden Texte direkt bearbeiten

### Icons/Emojis
Die Emojis in `index.html` sind einfach austauschbar:
- `🌱` = Pflanze
- `🎮` = Game
- `📊` = Stats
- etc.
